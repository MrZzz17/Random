# Phase 2: How to Send DMCA Takedown Notices

This guide walks through the practical steps of submitting your DMCA takedown notice (from `04_DMCA_TEMPLATE.md`) to each platform where your content appears.

---

## General Process

For **each site** in your tracking spreadsheet:

1. Find the site's DMCA / abuse contact (see directory below)
2. Fill in the DMCA template with that site's specific URLs
3. Send via the site's preferred method (web form, email, or both)
4. Save confirmation (screenshot the submission page, save the confirmation email)
5. Record the date filed in your tracking spreadsheet
6. If no response within 7 days, send the follow-up template (Appendix B in `04_DMCA_TEMPLATE.md`)

---

## Finding a Site's DMCA Contact

If a site is not listed below:

1. Scroll to the **bottom of the website** and look for links labeled "DMCA", "Copyright", "Legal", "Report Abuse", or "Contact"
2. Check the site's **Terms of Service** or **Privacy Policy** for a designated DMCA agent
3. Search Google: `"[site name]" DMCA contact` or `"[site name]" content removal`
4. Look up the **hosting provider** via WHOIS:
   - Go to https://whois.domaintools.com/ or https://who.is/
   - Enter the domain name
   - Find the hosting company and contact their abuse department
5. Check the **US Copyright Office DMCA Agent Directory**: https://www.copyright.gov/dmca-directory/

## Major Adult Site DMCA Directory

### Pornhub (and MindGeek / Aylo network sites)

Covers: Pornhub, YouPorn, RedTube, Tube8, Brazzers, and other Aylo properties.

- **Web form (preferred)**: https://www.pornhub.com/content-removal
- **Email**: contentremoval@pornhub.com
- **Response time**: Typically 24-48 hours
- **Notes**: Requires photo ID verification. Since December 2020, all content must be from verified uploaders, making unauthorized content easier to remove.

### XVideos

- **Web form**: Use "Report" button on each video, or the DMCA page in the site footer
- **Email**: dmca@xvideos.com
- **Response time**: Typically 24-72 hours
- **Notes**: Provide exact URLs. They respond faster to email with multiple URLs batched.

### XNXX

- **Web form**: DMCA link in site footer
- **Email**: dmca@xnxx.com (owned by same company as XVideos — WGCZ)
- **Response time**: 24-72 hours

### XHamster

- **Web form**: https://xhamster.com/content-removal

### CamWhores / RecTube / CamCaps / Similar Aggregators

These sites are often harder to deal with as they may not respond:

- **Step 1**: Look for any DMCA or contact page on the site
- **Step 2**: If no response, look up the hosting provider via WHOIS and file with them
- **Step 3**: If behind Cloudflare, file an abuse report at https://abuse.cloudflare.com/ to get the real hosting provider

### Reddit

- **Non-consensual intimate images**: https://reddit.zendesk.com/hc/en-us/requests/new?ticket_form_id=16374486893588

### Twitter / X

### Telegram

- **Email**: abuse@telegram.org or dmca@telegram.org
- **In-app**: Long-press message > Report
- **Response time**: Variable (can be slow)
- **Notes**: For channels / groups distributing content, report the channel itself.

### OnlyFans (if content is reposted there)

- **DMCA form**: https://onlyfans.com/dmca
- **Email**: dmca@onlyfans.com
- **Response time**: 24-48 hours

## If a Site Is Unresponsive

If a site does not respond within 7-10 business days after your follow-up:

1. **File with the hosting provider** — use WHOIS to identify the host, then find their abuse email (usually `abuse@[host].com`)
2. **File with Cloudflare** if the site uses it — https://abuse.cloudflare.com/ — Cloudflare will forward to the host and may reveal the real server IP
3. **File with the domain registrar** — found via WHOIS — some registrars will suspend domains for DMCA violations
4. **File for search engine de-indexing** (see `06_SEARCH_ENGINE_DELISTING.md`) — even if the content stays up, removing it from search results dramatically reduces visibility
5. **Escalate** (see `08_ESCALATION_GUIDE.md`)

## Batch Sending Tips

- Group all URLs for the same site into **one notice** rather than sending separate emails per URL
- Use the **full DMCA template** for email, the **short version** for web forms with character limits
- Send during **business hours** of the platform's time zone for faster processing
- Keep a copy of **every notice sent** — you may need them if you escalate
