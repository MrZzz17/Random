# Phase 3: Search Engine De-indexing

Even after content is removed from the original site, it can persist in search results via cached pages, thumbnails, and previews. Filing for de-indexing removes these traces from search results.

**Important**: De-indexing does not delete the content from the source website. It only removes it from search results. Always file DMCA takedowns with the hosting site first (Phase 2), then de-index.

---

## Google

Google offers two separate removal paths. Use **both** for maximum coverage.

### Path 1: Non-Consensual Intimate Image Removal (Recommended First)

This is the fastest path and does not require proving copyright — only that the images are intimate and shared without consent.

1. Go to: https://support.google.com/websearch/troubleshooter/9685456
2. Select "I want to remove images of myself"
3. Follow the prompts for "intimate images shared without consent"
4. Provide:
   - The URLs of the Google search results showing the content
   - The URLs of the source pages
   - Your contact information
5. Google typically processes these within **1-3 business days**

### Path 2: DMCA Copyright Removal

Use this in parallel, especially for video content or if Path 1 doesn't cover everything.

1. Go to: https://www.google.com/webmasters/tools/dmca-dashboard
2. Click "Create a new notice"
3. Fill in:
   - **Type of work**: Audiovisual / Video or Image
   - **Original work URL**: Your profile page on the cam platform (as proof of ownership)
   - **Infringing URLs**: Each Google search result URL that shows the content
4. Sign the declaration
5. Submit
6. Processing time: **1-7 business days**
7. Track status at the DMCA dashboard

### Google Cache Removal (after source content is deleted)

Once the source site has removed the content, force Google to update its cache:

1. Go to: https://search.google.com/search-console/removals (requires Google account)
2. Or use: https://www.google.com/webmasters/tools/removals
3. Enter the URL of the page that was removed
4. Request "Remove outdated content"
5. This clears cached versions within **24-48 hours**

## Bing

### DMCA Removal

1. Go to: https://www.bing.com/webmaster/tools/contentremoval
2. Or use the DMCA report form: https://www.microsoft.com/en-us/concern/dmca
3. Provide:
   - The Bing search result URLs
   - The infringing source URLs
   - Your ownership information
4. Processing time: **3-5 business days**

### Non-Consensual Image Removal

1. Go to: https://www.microsoft.com/en-us/concern/revengeporn
2. Fill out the form with the URLs and your information
3. This covers Bing search results

## Yandex

Particularly important since the client is Russian and Yandex is the dominant search engine in Russia.

### Content Removal Request

1. Go to: https://yandex.com/support/abuse/troubleshooting/photo-video.html
2. Or: https://yandex.com/support/abuse/
3. Select the appropriate complaint type (copyright violation or personal data)
4. Provide:
   - Yandex search result URLs showing the content
   - Source page URLs
   - Your identity information

### Russian Data Protection Law (Federal Law 152-FZ)

Yandex is subject to Russian law. Under 152-FZ, you can request removal of personal data:

1. File a complaint directly with Yandex citing 152-FZ
2. If Yandex does not comply, file with **Roskomnadzor** (see `08_ESCALATION_GUIDE.md`)
3. Roskomnadzor can order Yandex to de-index the content

---

## DuckDuckGo

DuckDuckGo sources results from Bing, so Bing removals often propagate. However, you can also file directly:

1. Email: dmca@duckduckgo.com
2. Include the same information as your standard DMCA notice

---

## Other Search Engines

- **Yahoo**: Uses Bing's index — Bing removals typically propagate to Yahoo
- **Baidu**: If content appears on Chinese search results, file at https://www.baidu.com/duty/right.html

---

## After Filing: Verification

After filing de-indexing requests:

1. **Wait the stated processing time** (1-7 days depending on the engine)
2. **Search again** using the same terms you used during evidence gathering
3. **Verify** that the results no longer appear
4. If results persist, **re-file** or escalate
5. Update the tracking spreadsheet with de-indexing status

---

## Tips

- File with **all** search engines, not just the one where you found the content — it likely appears on multiple
- De-indexing requests for **non-consensual intimate images** are processed faster than standard DMCA requests
- If the source site removed the content but Google still shows it, use the **cache removal tool** (not DMCA) — it's faster
- Keep the DMCA dashboard bookmarked — you can track all your requests from one place
